# Python Progress

This repository documents my journey of learning Python, starting from basic beginner projects and gradually moving toward more advanced programming concepts and libraries.

It includes simple projects such as calculators and small terminal-based programs, as well as later projects where I explored libraries like pandas and scikit-learn. Through these projects, I have practiced working with data and applying machine learning concepts in Python.

The purpose of this repository is to track my progress, organize my projects, and show how my skills have developed over time.
